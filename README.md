## Getting Started

1. Clone the repository
```bash
git clone https://github.com/chandu55526/Chandraportfolio.git
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Run tests
```bash
npm test
```

5. Build for production
```bash
npm run build
```
